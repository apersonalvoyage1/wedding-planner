How do I get this URL? (setup guide)
Step 1: Create a Google Sheet with two tabs: Expenses and Guests
Step 2: Add headers as described in the google-apps-script.js file
Step 3: Go to Extensions → Apps Script, paste the script code
Step 4: Deploy → New deployment → Web app
Step 5: Set "Execute as" = Me, "Who has access" = Anyone
Step 6: Copy the deployment URL and paste it above
